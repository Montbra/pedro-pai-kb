# Octopus Invaders — Projeto

Vertical space shooter com vanilla JS, canvas pixel art.

## Documentação

- [[Guia de Implementação]] — Guia passo a passo (6 fases) otimizado para agentes 9B

## Referências

- Código: `~/octopus-invaders/index.html`
- Servidor de teste: `python3 -m http.server 3001`

## Fases

1. Fundação estática (canvas fullscreen, config)
2. Motor e jogador (loop, lerp 0.35)
3. Arte pixelada e inimigos (fillRect grids)
4. Combate, tiros e colisões (circle collision)
5. Chefe e modo Unleash
6. UI, HUD e polimento
